# Entrega2-PaginaWeb
Proyecto Transito

La secretaría de transito de Sabaneta lo contrata para desarrollar una solución web que utilice el Modelo por capas (Modelo Vista Controlador)  y utilice al menos una de estas estructuras: ARRAYLIST, archivo, arreglo de objetos o Base de Datos.
Para que un vehículo transite debe haber una matrícula donde se requiere información necesaria como placa, marca, fecha de matrícula del vehículo y propietario. El propietario puede ser una persona o una empresa, donde ambos fundamentalmente tienen una identificación, un nombre y una dirección. Una persona puede tener matriculados varios vehículos. Un vehículo debe ser propiedad de un solo propietario. Un vehículo puede ser de tres tipos: automóvil, moto o carro pesado. Cuando un vehículo obtiene una infracción, se registra en una fecha y pudo ser accionada por un agente de tránsito o por una cámara de detecciones.
Se requiere:
Realizar el diagrama de clases. (5 puntos)
Crear CRUD sobre la información referenciada anteriormente.(20 puntos)
Desarrollar un reporte sobre infracciones y/o propietarios.(15 puntos)
Aprendizaje independiente. Puede ser una funcionalidad adicional (10 puntos).
